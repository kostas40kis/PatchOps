# PatchOps LLM Packet

This folder contains the replacement files intended to be pasted into or uploaded for future LLM work.

## Files

- `active_continuation_prompt.txt` — use when work is already in progress and handoff artifacts exist
- `new_target_bootstrap_prompt.txt` — use when starting a brand-new target repo with PatchOps
- `project_packet_template.md` — starter template for `docs/projects/<target>.md`
- `generic_packet_file_list.md` — the exact generic docs to upload first for new-target onboarding

## Rule of thumb

- continuation starts with handoff
- new-project startup starts with the generic packet and then a project packet
