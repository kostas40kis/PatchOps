# Examples

## Maintained examples

Classic examples should still demonstrate:

- `check`
- `inspect`
- `plan`
- `apply`
- `verify`

Bundle examples should still demonstrate:

- `check-bundle`
- `run-package`

## Example guidance

Examples should reflect the current modular package layout, the one canonical truth rule, the suspicious-success blocker wording, and the code-first / docs-last development process summary.
